
# sudo awakening Website for Squarespace (Manual Copy Setup)

## How to Use

1. Go to your Squarespace admin panel.
2. Choose the Brine or Bedford template.
3. Create the following sections/pages:
   - Home
   - About
   - Projects
   - Blog
   - Contact

## Add Custom Code

- Copy contents of `style.css` into **Design > Custom CSS**.
- Use the content from `index.html` in **Code Blocks** on each page.

## Fonts Used

- JetBrains Mono (for code look)
- Inter (for body text)
